About PhishGuard AI

PhishGuard AI is an intelligent cybersecurity tool designed to protect users from phishing emails, SMS scams, and malicious URLs. It uses a multi-layer detection engine combining pattern recognition, NLP heuristics, and a curated scam knowledge base.

This project was built as an AI-powered web application for the Security domain. The system requires no external API calls — all analysis runs locally in the browser using our custom ML classifier engine.

How It Works

1. Pattern-Based ML Classifier: Analyzes text against 50+ known phishing patterns including urgency tactics, threat language, credential harvesting attempts, and suspicious URLs.

2. RAG Knowledge Base: A retrieval-augmented knowledge base of known Pakistani and international scam patterns (HBL spoofing, FBR phishing, fake prize scams, etc.) is embedded directly into the classifier.

3. URL Analysis Engine: Checks for suspicious TLDs, brand impersonation, IP-based URLs, HTTP vs HTTPS, subdomain abuse, and known safe domains.

4. Scoring System: Each signal adds or subtracts from a 0-100 risk score. Threshold-based verdict: 0-29 Safe, 30-59 Suspicious, 60-100 Dangerous.
